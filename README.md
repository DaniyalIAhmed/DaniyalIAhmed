Hi 👋 My name is Daniyal Ahmed
==============================

Software Developer
------------------

I began my journey into web development in late 2022, starting with the foundational technologies—HTML, CSS, and JavaScript. As I continued to build small projects and deepen my understanding, I gradually progressed from basic front-end skills to more advanced JavaScript development.

By 2024, I had not only mastered JavaScript but also expanded my skill set to include TypeScript, which significantly improved the scalability and maintainability of my code. Around this time, I transitioned from working on tutorial-based or experimental projects to building real-world applications that address actual problems and deliver tangible value.

Most recently, I completed work on Devpulse, a web-based code editor inspired by the VibeCode design philosophy. Devpulse stands out by addressing a major challenge in modern development workflows—access to expensive LLM (Large Language Model) APIs for features like code suggestions and autocompletion.

Instead of relying on costly third-party API keys, Devpulse leverages the user's own machine resources to run local models, offering AI-powered coding assistance without the ongoing financial burden. This not only makes AI-assisted development more accessible to a wider range of users but also ensures greater privacy and control over data.

With this project, I aimed to bridge the gap between powerful AI tools and independent developers or small teams who may not have the budget for premium LLM APIs. Devpulse reflects my growth as a developer, combining my technical skills with a product-oriented mindset to create something that genuinely solves a problem.
